Examples of CSS code.

Text colour:

Colour of text can be changed by simply adding the color tag either in another tag such as in <h1 color: lightblue>Hello World!<h1> or separately in the <style> tag

Colur can be described as a word which includes many examples such as lightblue or green but can also be described in rgb color: rgb(13,255,0) or even hexadecimal color: #0DFF05

Font weight

Font weight refers to the thickness of the font. This can be changed by simply adding the font-weight tag either in another tag such as in <h1 style = "font-weight:bold">Hello World!<h1> or, as before separately in the <style> tag.

Font weight can be described either as a word such as bold or bolder or as a value such as 700 which is equivalent to bold
